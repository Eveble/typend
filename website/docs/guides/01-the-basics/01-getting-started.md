[typend][package-homepage]

Runtime validation for TypeScript

## Requirements

- [Node.js][nodejs] MUST be v14.0 or later.

## Installation

To use typend with your Eveble app:

```bash
npm install typend
```

or

```bash
yarn add typend
```

[nodejs]: https://nodejs.org/
